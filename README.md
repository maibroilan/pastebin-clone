# Pastebin Clone

A Pastebin clone with Go and SvelteKit.

## Features

- Create pastes
- Expiration dates
- Optional password protection
- Rate limiting
- Shareable links

## Tech Stack

### Frontend

- SvelteKit
- TypeScript
- Tailwind CSS
- Bun

### Backend

- Go
- Chi
- PostgreSQL
- pgx
- sqlc
- Redis
